# AWS Notes

A personal repository for learning and documenting Amazon Web Services (AWS).

## Topics Covered

- AWS Global Infrastructure
- IAM (Identity and Access Management)
- EC2 (Elastic Compute Cloud)
- S3 (Simple Storage Service)
- VPC (Virtual Private Cloud)
- RDS (Relational Database Service)
- Lambda
- CloudWatch
- Route 53
- ECS & EKS
- Security Best Practices
- Cost Optimization
- AWS Architectures

## Purpose

This repository serves as a centralized place to:

- Store AWS learning notes
- Document hands-on labs
- Track AWS concepts and services
- Save architecture diagrams and examples
- Prepare for AWS certifications
